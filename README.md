# Mosca-MQTT Broker

Arquivos básicos do broker para armazenar dados dos sensores na collection do MongoDB.

## Pacotes necessários

```JS
    mongodb: 3.5.8,
    mosca: 2.8.3,
    mqtt: 4.1.0
```
